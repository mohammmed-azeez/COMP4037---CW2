# NHS Hospital Admissions — Post Pandemic Recovery Analysis

An interactive data visualisation web application built with Flask and Plotly, analysing 26 years of NHS England hospital admissions data (1998–2024) across 264 diagnosis categories. The app reveals which conditions remain trapped in the post-COVID backlog categories where admissions are still below pre pandemic levels while waiting lists have grown through five advanced chart types: a recovery heatmap, parallel coordinates plot, treemap, radar chart, and scatterplot matrix, all using the 2018-19 financial year as a pre-pandemic baseline. Built for COMP4037 Research Methods CW2, University of Nottingham.
